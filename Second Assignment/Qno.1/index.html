Differences between margin and padding are in the table below:

Margin Padding
Margin is the space around an element’s border Padding is the space between an element’s border and the element’s content
Margin property controls the space outside an element Padding property controls the space inside an element
Margins can be used to: i)Change an Element’s Position on the Page ii)Set the Distance Between Nearby Elements iii)Overlap Elements Padding can be used to: i)Add Space Between Content and Its Border ii)Change the Size of an Element
For example, if you want 30 pixels on the top, 75 pixels on the right, 45 pixels on the bottom, and 100 pixels on the left, you’d use: p { margin: 30px 75px 45px 100px; } For example, if you want 30 pixels on the top, 75 pixels on the right, 45 pixels on the bottom, and 100 pixels on the left, you’d use: p { padding: 30px 75px 45px 100px; }
